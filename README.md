# python module
A python module for munkireport

> v. 1.1.0 
> November 19, 2025 
> Alex Narvey / Precursor.ca  

![Python Module Report](BlueSky_module.png)

Reports on various installations of python including:

* Xcode python
* homebrew python
* munki python
* munkireport python (versions 2 and 3)
* autopkgr python

Also shows the version of python and where it is located in the file system.
